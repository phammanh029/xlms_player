# ebook_player
    play a zip file contain image and audio by name index
# Get started
    1. prepare a zip file with name of image and mp3 files follow format: ${name}.(png|jpg) ${name}.mp3

    example:

    content:
        1.png
        1.mp3
        2.png
        3.mp3

    2. copy to phone
    3. open app
    4. select file
    5. drag image and listen to the audio
